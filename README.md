# J.P. Morgan Quantitative Research Virtual Experience

This repository contains the tasks and projects completed as part of the J.P. Morgan Chase & Co. Quantitative Research Virtual Experience Program hosted on Forage.

## Overview

The simulation focused on solving real-world quantitative research and financial analytics problems using Python, statistics, and data analysis techniques.

Through this program, I gained practical exposure to:

- Quantitative research workflows
- Financial data analysis
- Commodity contract pricing
- Credit risk analysis
- FICO score segmentation
- Predictive modeling concepts
- Python programming for finance

---

## Tasks Completed

### 1. Investigate and Analyze Price Data
- Explored and analyzed historical price datasets
- Identified patterns and trends in commodity pricing
- Derived insights for quantitative decision-making

### 2. Price a Commodity Storage Contract
- Built a pricing model for commodity storage contracts
- Applied quantitative finance concepts
- Estimated profitability based on storage and pricing dynamics

### 3. Credit Risk Analysis
- Analyzed loan portfolio data
- Estimated probability of default
- Evaluated financial risk using analytical techniques

### 4. Bucket FICO Scores
- Implemented FICO score bucketing using dynamic programming
- Converted continuous credit scores into categorical segments
- Improved risk modeling and default prediction
